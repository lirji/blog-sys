# 石墨文档批量导入

把石墨导出的 Markdown 放进这个目录，然后运行：

```bash
npm run db:import
```

## 用法要点

- **一篇文档 = 一个 `.md` 文件**。石墨里单篇「··· → 导出为 → Markdown」。
- **子文件夹名 = 分类**。例如 `db/import/后端/xxx.md` → 分类「后端」；直接放根目录 → 「未分类」。
- **标题**：优先用文档里的第一个 `# 一级标题`，没有就用文件名。
- **图片**：自动处理三种引用——远程 URL（下载）、同目录相对路径（如石墨导出的 `images/xx.png`）、base64 内联；全部存入数据库并改写为 `/media/<id>`。私有/需登录的远程图会跳过并在末尾报告。
- **发布状态**：默认直接发布。想先存草稿再逐篇审核：`IMPORT_AS_DRAFT=true npm run db:import`。
- **重复**：slug 由文件名生成，和已有文章冲突时自动加后缀；重复运行同一批文件会产生副本，导入完记得清空本目录。

导入完成后，去 `/admin` 检查、在编辑器里微调即可。
